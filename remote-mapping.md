# Mapping of universal remote

Map Moncaso 312 to One For All Evolve 4

```
/---------\
|  x x x  | AV, Magic, Power
|  x x x  | 1, 2, 3
|  x x x  | 4, 5, 6
|  x x x  | 7, 8, 9
|  x x x  | Info, 0, Fav
|    x    | Up
|  x x x  | Left, Ok, Right
|    x    | Down
|  x x x  | Back, Activity, Exit
|  x x x  | Vol-Up, Menu, Chan-Up
|    x    | Guide
|  x x x  | Vol-Down, Smart, Chan-Down
|  x   x  | Mute, NetTV, PreviousProgram
| x x x x | Red, Green, Yellow, Blue
|  x x x  | Prev, Play, Next
|  x x x  | Rewind, Pause, Forward
|  x x x  | List, Rec, Stop
\---------/
```
